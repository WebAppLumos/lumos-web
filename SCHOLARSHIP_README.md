# Lumos Web 소스 코드 (Source Code)

이 디렉토리는 Lumos Web 프로젝트의 프론트엔드 소스 코드를 포함하고 있습니다.

## 📁 주요 구성 요소

- **Scholarship.jsx**: 장학금 큐레이션 서비스의 메인 로직과 UI가 구현된 핵심 컴포넌트입니다. 사용자의 프로필(성적, 소득구간 등)을 분석하여 맞춤형 장학금을 추천합니다.
- **Scholarship.css**: 장학금 페이지 전용 스타일 시트입니다.
- **certifications.json**: 장학금 수혜 조건 확인에 사용되는 자격증 데이터베이스입니다.
- **firebase.js**: Firebase 인증 및 데이터베이스 연동을 위한 설정 파일입니다.
- **main.jsx**: 어플리케이션의 진입점(Entry Point)입니다.

## 🛠️ 주요 기능

1. **사용자 프로필 관리**: 전공, 학년, GPA, 취득 학점, 소득구간 등을 입력 및 수정할 수 있습니다.
2. **장학금 자격 자동 확인**: 입력된 프로필을 바탕으로 국가장학금, 교내 성적우수 장학금, 자기계발 장학금 등의 수혜 가능 여부를 즉시 계산합니다.
3. **자격증 및 어학 성적 연동**: 토익 점수 향상이나 특정 자격증 취득에 따른 장학금을 추천합니다.
